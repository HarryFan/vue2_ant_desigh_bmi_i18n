BMI 計算器
=======

這是一個使用 Vue 2 和 Ant Design Vue 實現的多語言 BMI 計算器應用。用戶可以輸入身高和體重，並在選擇不同語言時查看結果。支持的語言包括繁體中文、簡體中文、英文和韓文。

功能
--

*   輸入身高和體重
    
*   計算 BMI（體重指數）
    
*   支持多語言（繁體中文、簡體中文、英文和韓文）
    
*   在右上角切換語言
